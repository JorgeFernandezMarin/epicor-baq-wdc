Epicor BAQ → Tableau (via Azure Function)

Este conector NO usa usuario/contraseña. La autenticación hacia Epicor se hace dentro de la Azure Function.

1. URL de la Azure Function

Usa la URL completa de la función (incluyendo el ?code=...). Ejemplo:
https://func-epicor-ripipsa-test-h2bha3hzcmeagwe3.centralus-01.azurewebsites.net/api/EpicorBAQProxy?code=...

2. Parámetros del BAQ

Este campo se utilizará para las extracciones incrementales en Tableau (ej. InvcHead_CreatedOn).
